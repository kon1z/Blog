# 🤣阿星Plus⭐⭐⭐ 个人博客

## 项目介绍
本项目基于[阿星Plus个人博客项目](https://github.com/Meowv/Blog/tree/main)进行的二次开发，使用.NET9、abp vNext 9.*版本进行开发。与原版不同的是想尽可能的遵循abp以及DDD最佳实践来进行二次开发。可作为.NET Core中高级项目进行学习。
